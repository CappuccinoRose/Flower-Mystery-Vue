# 花之谜语 - Flower Mystery

一个关于花卉的网站，提供花卉搜索、筛选、收藏、送花指南和文化故事等功能。

## 项目功能

- **用户系统**：登录、注册、个人中心、收藏管理
- **花卉搜索与筛选**：按名称、花期、起源排序，按标签筛选
- **送花指南**：12个常见场景（生日、婚礼、乔迁等）及其推荐花卉
- **故事专栏**：12个花卉文化故事（玫瑰、莲花、郁金香等）
- **响应式设计**：适配各种设备屏幕

## 技术栈

- **前端框架**：Vue 3 + Vue Router
- **状态管理**：Pinia
- **UI组件**：Feather Icons
- **样式**：CSS3 + Flexbox + Grid
- **图片懒加载**：Intersection Observer API

## 安装与运行

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/CappuccinoRose/Flower-Mystery-Vue.git
   ```

2. 进入项目目录：
   ```bash
   cd flower-mystery
   ```

3. 安装依赖：
   ```bash
   npm install
   ```

4. 启动开发服务器：
   ```bash
   npm run serve
   ```

5. 打开浏览器访问：[http://localhost:5173](http://localhost:5173)
